# Azure Data Collector - Copilot Agent Instructions

## プロジェクト概要
Azure顧客からAzureリソース情報を収集し、Firebaseでデータを管理するWebアプリケーション。
Firebase Authentication + Azure AD認証によりセキュアにAzureデータを取得・表示する。

## アーキテクチャ
- **フロントエンド**: React + TypeScript + Material-UI + Vite
- **バックエンド**: Python Firebase Functions + Azure SDK
- **データベース**: Firestore (リアルタイム同期)
- **認証**: Firebase Auth + Azure AD (MSAL)
- **デプロイ**: Firebase Hosting + Cloud Functions

   - 要件: Azure認証情報必須

### Firestore データ構造
```
users/{userId}/
├── resources/          # Azureリソース一覧
├── costs/             # コスト情報
├── subscriptions/     # サブスクリプション情報
└── tenants/{tenantId}/
    ├── resources/     # テナント別リソース
    ├── costs/        # テナント別コスト
    └── metadata/     # テナント情報
```

## 現在の課題と対処法

### 🚨 最優先課題: ダッシュボードのデータ表示
**症状**: テストデータは表示されるが、実データが0件
**原因**: Azure認証情報未設定

**対処手順**:
1. テストデータ生成で基本動作確認
2. Firestoreデータ存在確認
3. Azure認証情報設定

### 🔍 デバッグ方法
```bash
# フロントエンド開発サーバー起動
cd frontend && npm run dev

# Firebase Functions ログ確認
firebase functions:log --only collect_azure_data

# Firebaseデプロイ
firebase deploy --only functions
firebase deploy --only hosting
```
